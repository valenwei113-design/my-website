# Valen Wei

Personal homepage — a visual journal of travel, life, and learning.

## What's here

- **Home** — Immersive full-screen video slideshow with ambient music, cycling through moments from Iceland, Norway, diving trips, and more.
- **Journey** — A collection of long-form stories in four chapters:
  - Living abroad across different countries
  - A road trip through Norway
  - Studying in Germany — from university types to visa costs
  - Early career reflections across state-owned and foreign enterprises
- **Vlog** — Curated travel clips and daily moments.
- **Xiaohongshu** — Social cross-post highlights.

## Tech

Plain HTML + Tailwind CSS, hosted on GitHub Pages.
